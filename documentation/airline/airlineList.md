**List Airline**
----
  Returns json data for list of airlines.

* **URL**

  /rs/airline

* **Method:**

  `GET`
  
*  **URL Params**

   None

* **Data Params**

  None

* **Success Response:**

  * **Code:** 200
  
    **Content:** 
    ```
    [
        {
            "id": 2,
            "name": "Ramesh"
        }
    ]
    ```
 
* **Error Response:**

  None
