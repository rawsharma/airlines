**Update Airline**
----
  Update an airline

* **URL**

  /rs/airline/:id

* **Method:**

  `PUT`
  
*  **URL Params**

   **Required:** `id=[integer]`

* **Data Params**

  **Required:** `name=[string]`

* **Success Response:**

  * **Code:** 200

 
* **Error Response:**

  * **Code:** 500 Server Error
