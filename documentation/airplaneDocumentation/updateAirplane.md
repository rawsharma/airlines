**Update Airplane**
----
  Update an airplane

* **URL**

  /rs/airplane/:id

* **Method:**

  `PUT`
  
*  **URL Params**

   **Required:** `id=[integer]`

* **Data Params**

  **Required:** `capacity=[integer]`
    
  **Required:** `model=[string]`
    
  **Required:** `serialnr=[string]`

* **Success Response:**

  * **Code:** 200
 
 
* **Error Response:**

  * **Code:** 500 Server Error
