**Create Airport**
----
  Create an airport

* **URL**

  /rs/airport

* **Method:**

  `POST`
  
*  **URL Params**

   None

* **Data Params**

  **Required:** `airportcode=[string]`
  
  **Required:** `city=[string]`
  
  **Required:** `country=[string]`
  
  **Required:** `name=[string]`

* **Success Response:**

  * **Code:** 200
  

 
* **Error Response:**

  * **Code:** 500 Server Error
